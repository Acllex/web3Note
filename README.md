## 项目所使用的技术

- vue 全家桶
- markdown-it 显示文本
- 使用 **节流** 技术实现文档自动保存
- 二次封装 axios,实现请求统一化管理
- less 进行 css 页面布局
- es6 语法进行 js 开发
- ant-design-vue 组件库开发样式

## 安装项目依赖

`yarn install`

### 运行项目

`yarn serve`

### 打包部署

`yarn build`
